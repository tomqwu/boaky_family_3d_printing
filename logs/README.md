# Build Logs

One file per week: `week-1.md`, `week-2.md`, … copy the template from
[`../program/07-build-log-template.md`](../program/07-build-log-template.md).

Why bother? Two reasons:

1. Looking back at week 1 in week 8 is the most satisfying part of the summer.
2. Failed prints you wrote down become "we know how to fix that" forever.

Drop photos in a `photos/` subfolder per week if you want (`logs/week-1/photos/`).
